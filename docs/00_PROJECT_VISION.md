# Project Vision

- **Status:** Approved baseline
- **Last Updated:** 2026-07-18
- **Authority:** 제품 정체성과 원칙의 최상위 문서
- **Purpose:** 전면 개편에서 바뀌지 않는 제품 정의와 성공 기준을 고정한다.

## 정의

Dechive는 **사람이 직접 공부하고 이해한 내용을 지식, 강의, 실습, AI 업데이트로 다시 구성하는 독립적인 학습 플랫폼**이다.

사용자는 개념을 읽고, 구조화된 강의를 따라가고, 직접 실행하며, 빠르게 변하는 AI 정보를 확인한다. 신뢰는 “검증”이라는 광고 문구가 아니라 출처, 마지막 확인일, 수정 이력, 실제 실행 결과로 드러난다.

## 제품 약속

- 복잡한 개념은 쉽게 설명하되 근거와 한계를 숨기지 않는다.
- 지식→강의→실습→업데이트가 관련 콘텐츠로 연결된다.
- 콘텐츠와 미디어의 원본·운영 권한은 Dechive가 가진다.
- AI는 제작을 보조하며 사람의 승인 없이 발행하지 않는다.
- 공개 URL과 기존 콘텐츠는 계획적으로 이전하고 검증한다.

## 아닌 것

일반 블로그, Markdown/Obsidian 발행 사이트, AI SaaS 랜딩페이지, 뉴스 수집기, 커뮤니티, 영상 목록, 물리적 도서관 모사, 미래형 AI 비주얼 사이트가 아니다. 외부 채널은 배포 수단이지 정보 구조가 아니다.

## 성공 기준

1. 독자가 한 주제를 이해하고 실제로 재현할 수 있다.
2. 모든 핵심 콘텐츠에서 출처·확인일·수정 이력을 추적할 수 있다.
3. 운영자가 사이트 내부에서 초안부터 발행·수정·SEO·미디어까지 관리한다.
4. 기존 검색 유입 URL을 보존하거나 검증된 redirect를 제공한다.
5. 데이터·미디어를 표준 형식으로 내보내고 공급자를 교체할 수 있다.
