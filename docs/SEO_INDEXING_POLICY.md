# SEO and Indexing Policy

sitemap에는 새 공개 canonical URL과 실제 published Knowledge/Lecture detail만 포함한다. admin, draft, preview, unpublished, redirect, 404/410, legacy URL은 포함하지 않는다.

legacy URL은 robots 차단 대신 410 Gone으로 응답해 검색 엔진이 삭제 상태를 확인할 수 있게 한다. canonical은 production domain `https://dechive.dev`의 자기 URL을 사용하며 Preview domain을 사용하지 않는다.
